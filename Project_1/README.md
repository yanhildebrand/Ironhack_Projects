#### YouTube Analytics Dashboard

Provides a visual overview over the currently trending Youtube videos and tags per Country and per Category
for youtube creators to optimize their content according to the current trends.

Filters: Country, Category.
Metrics: number of views, number of likes, number of comments, tags.
Results: 3 top-most trending videos in terms of the current metric (views, likes, comments) or the most common tags.
